# 🐾 Zoorofile

> **귀여운 동물 캐릭터로 꾸미는 GitHub Profile README 생성기**

나만의 동물 캐릭터를 선택하고, 자동으로 업데이트되는 귀여운 GitHub 프로필을 만들어보세요!

---

## ✨ 기능

-   **원하는 동물 캐릭터** 선택
-   **커밋 활동에 따라 달라지는 동물의 상태**
-   **레포지토리별 이번 주 기여 섹션 표시**
-   **언어 설정 (한국어, 영어 지원)**
-   **매일 자동 업데이트** (GitHub Actions)
